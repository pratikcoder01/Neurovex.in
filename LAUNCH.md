## 🎯 How to Launch Neurovex

### **Step 1: Open Main Application**
```
📂 Neurovex/
└── 📄 index.html
```
Double-click `index.html` or open in your browser

### **Step 2: Navigate to Features**
- **🏠 Home**: `index.html` - Main landing page
- **📊 Dashboard**: `pages/dashboard.html` - Real-time EEG monitoring
- **🎮 Hardware**: `pages/hardware.html` - Control interface
- **⚙️ Settings**: `pages/settings.html` - System configuration

### **Step 3: Start Using**
All features work immediately:
- **📊 Dashboard** - Simulated EEG monitoring
- **🎮 Hardware** - Manual and mind control testing
- **⚙️ Settings** - System configuration
- **🧠 Calibration** - Personalized settings

No hardware required - everything works out of the box!
